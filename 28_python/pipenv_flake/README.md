# SIMPLE CLI SKELETON

Demonstrate a simple skeleton that other projects can be based on.  

Demonstrates:

* flake8 linting
* logging
* argument parsing
* docstrings

## Development using nix

Open a pure shell and install packages

```sh
nix flake show templates

nix flake new pipenv_flake

# enter folder 
cd ./pipenv_flake

nix build 

nix flake metadata

nix flake check

nix develop --impure --command bash -c 'python --version'

# enter the flake
nix develop
```

## Start

```sh
# this is now in the flake
export PIPENV_VENV_IN_PROJECT=1
# install
pipenv install --dev

# see the packages inside the venv
pipenv run pip list

# create .env file
cp .env.template .env
. ./.env

# lint and test code
pipenv run lint
pipenv run test

# enter venv
pipenv shell

# run with arguments
pipenv run start --test
pipenv run start:test 
```

## Debugging and Troubleshooting

```sh
# enter python
pipenv run python

> import main

> main.test.__doc__
```
